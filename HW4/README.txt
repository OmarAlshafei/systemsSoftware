To compile & run this code, first download the hw4compiler.c file and any requisite
input files to the same directory.
Use "gcc -g hw4compiler.c" to generate a program file from the .c file.

You can now run the program while passing in your input file with this
command: "./a.out input.txt".

The expected output will be shown directly in the terminal.
